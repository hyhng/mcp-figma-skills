Co to je What is it

Figma Skills pro MCP jsou předpřipravené instrukce, které naučí tvého AI asistenta (Claude Code, Cursor…) jak spolehlivě zvládat běžné Figma úkoly. Místo abys pokaždé popisoval každý krok, stačí říct „implementuj tento design" nebo „propoj komponenty s kódem" — a AI ví přesně, co dělat. Funguje to jako obousměrný most mezi Figmou a kódem.

Figma Skills for MCP are pre-built instructions that teach your AI assistant (Claude Code, Cursor…) how to handle common Figma tasks reliably. Instead of describing every step each time, just say "implement this design" or "connect components to code" — and the AI knows exactly what to do. It works as a two-way bridge between Figma and code.

DESIGN → KÓD / CODE
figma-implement-design

Vezme Figma URL → přečte design → vygeneruje funkční kód s existujícími komponentami tvého projektu. Před odevzdáním si dokonce zkontroluje výstup oproti originálnímu designu.

Takes a Figma URL → reads the design → generates working code using your project's existing components. Even checks its output against the original design before finishing.

DEV MODE
figma-code-connect-components

Propojí Figma komponenty s jejich implementacemi v kódu. V Dev Mode pak vývojář klikne na komponentu a rovnou skočí do zdrojáku — žádné hledání.

Connects Figma components to their code implementations. In Dev Mode, a developer clicks a component and jumps straight to the source — no hunting required.

Org / Enterprise
SETUP
figma-create-design-system-rules

Jednou naučí AI konvence vašeho projektu — kde jsou komponenty, jaké tokeny používáte, jak se pojmenovávají soubory. Uloží je do configu a nemusíte to vysvětlovat pokaždé znovu.

Teaches the AI your project's conventions once — where components live, which tokens to use, how files are named. Saves it to a config so you never explain it again.

CANVAS WRITE
figma-use

Základní skill pro psaní přímo na Figma canvas. AI vytváří reálné, editovatelné objekty — framy, komponenty, proměnné, auto layout. Momentálně zdarma v beta.

The foundational skill for writing directly to the Figma canvas. AI creates real, editable objects — frames, components, variables, auto layout. Currently free in beta.

Beta · Free now
FIGJAM
figma-use-figjam

Stejná magie jako figma-use, ale pro FigJam. Přidává stickies, sekce, konektory, tabulky a diagramy přímo na board — ideální pro workshopy a plánování.

Same magic as figma-use, but for FigJam. Adds stickies, sections, connectors, tables, and diagrams directly to the board — perfect for workshops and planning.

UTILITY
figma-create-new-file

Vytvoří nový prázdný Figma soubor nebo FigJam board v tvojich draftech. Vrátí přímý odkaz. Jednoduchá věc, ale šetří čas při spouštění nových projektů přes AI.

Creates a new blank Figma file or FigJam board in your drafts. Returns a direct link. Simple, but saves time when spinning up new projects through AI.

Ukázkové Skills Example Skills

Plně funkční, ale také šablony pro vlastní tvorbu Fully functional, and also templates for building your own

KÓD → FIGMA
figma-generate-library

Vygeneruje nebo aktualizuje celou design system knihovnu z kódu. Prochází fázemi: Discovery → Foundations (tokeny, barvy) → File Structure → Komponenty → QA & Code Connect. Mezi fázemi čeká na tvůj review.

Generates or updates an entire design system library from code. Goes through phases: Discovery → Foundations (tokens, colors) → File Structure → Components → QA & Code Connect. Pauses for your review between phases.

KÓD → FIGMA
figma-generate-design

Sestaví celé obrazovky ve Figmě z existujícího kódu, pomocí reálných komponent a tokenů z design systému. U webapps umí udělat i screenshot živé aplikace jako referenci pro layout.

Builds full screens in Figma from existing code, using real components and tokens from the design system. For web apps, it can even take a screenshot of the live app as a layout reference.

Tipy pro design tým Tips for the design team

Jak z toho vytěžit maximum How to get the most out of it

01
Začni s nastavením pravidel
Start with setting up rules

Spusť figma-create-design-system-rules jednou na začátku projektu. Ušetříš si vysvětlování při každé další konverzaci s AI.

Run figma-create-design-system-rules once at project start. Saves explaining your conventions in every future AI conversation.

02
Sdílej konkrétní URL, ne celý soubor
Share specific URLs, not the whole file

Při použití figma-implement-design odkazuj přímo na konkrétní frame nebo komponentu — ne na celý soubor. AI bude přesnější.

When using figma-implement-design, link directly to the specific frame or component — not the whole file. The AI will be more precise.

03
figma-use je beta, ale zdarma
figma-use is beta, but free

Write-to-canvas přes AI bude do budoucna placené. Teď je to zdarma — ideální čas experimentovat a naučit se, kde to šetří čas.

Write-to-canvas via AI will be paid in the future. Right now it's free — the ideal time to experiment and learn where it saves time.

04
Code Connect = méně hledání pro devs
Code Connect = less hunting for devs

Investice do figma-code-connect-components se vyplatí celému týmu. Vývojáři přestanou hledat, která komponenta odpovídá jakému kódu.

Investing in figma-code-connect-components pays off for the whole team. Devs stop guessing which component maps to which code.

Jak začít How to get started

Možnost A — Plugin (doporučeno)

Option A — Plugin (recommended)

Pokud používáš Claude Code nebo Cursor, nainstaluj Figma plugin do svého nástroje. Skills se přidají automaticky — žádné ruční stahování.

If you use Claude Code or Cursor, install the Figma plugin into your tool. Skills are included automatically — no manual downloading.

Možnost B — Ruční instalace

Option B — Manual install

Stáhni skills z GitHubu a požádej AI asistenta, aby je rozbalil a umístil do správné složky.

Download skills from GitHub and ask your AI assistant to unzip them and place them in the right folder for your tool.
